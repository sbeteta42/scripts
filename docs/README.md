# Documentation

Ce dossier contient la documentation interne du dépôt `scripts`.

Il sert à centraliser les règles d’organisation, les conventions de nommage, la structure cible et les principes de maintenance du dépôt.

L’objectif est de garder une base claire pour :
- classer correctement les scripts ;
- maintenir une structure stable ;
- éviter le désordre à la racine ;
- documenter les choix d’organisation.

---

## Rôle du dossier `docs/`

Le dossier `docs/` regroupe les documents de référence liés au dépôt.

Il ne contient pas les scripts eux-mêmes, mais les éléments qui expliquent :
- comment le dépôt est organisé ;
- comment nommer les fichiers ;
- où ranger un nouveau script ;
- quelles règles appliquer pour garder un dépôt propre.

---

## Contenu recommandé

Exemples de fichiers attendus dans ce dossier :

- `README.md`
- `structure.md`
- `conventions.md`

Selon l’évolution du dépôt, on peut aussi y ajouter :
- une roadmap ;
- des notes de maintenance ;
- des règles de contribution ;
- des guides d’utilisation plus détaillés.

---

## Fichiers principaux

### `structure.md`
Décrit l’arborescence cible du dépôt et le rôle de chaque dossier.

### `conventions.md`
Définit les conventions de nommage, les règles de classement et les bonnes pratiques de maintenance.

---

## Pourquoi ce dossier est utile

Sans documentation, un dépôt finit souvent par devenir :
- difficile à relire ;
- incohérent ;
- encombré ;
- compliqué à faire évoluer.

Le dossier `docs/` permet de poser des règles simples et stables.

---

## Bonnes pratiques

- garder les documents courts et utiles ;
- éviter les doublons entre fichiers ;
- mettre à jour la documentation quand la structure évolue ;
- utiliser un style clair, direct et cohérent avec le reste du dépôt.

---

## Objectif

Le dossier `docs/` doit aider à répondre rapidement à ces questions :

- où ranger un script ?
- comment le nommer ?
- comment garder le dépôt propre ?
- quelle est la structure cible ?

Si ces réponses sont faciles à retrouver, la maintenance du dépôt devient beaucoup plus simple.
